# Awesome Claude Code Skills

A curated list of community skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Code Review

- [seven-lens-review](https://github.com/jkgdq789-blip/seven-lens-review) -- 7-lens deep code review (CITR, state transitions, security, boundary values, consistency, dead code, test gaps)
- [trace-interaction](https://github.com/jkgdq789-blip/trace-interaction-skill) -- Cross-component interaction scenario tracing
- [adversarial-audit](https://github.com/jkgdq789-blip/adversarial-audit-skill) -- Find real bugs by executing code, not reading (skip masking, catch swallowing, CI false greens)
- [verify-by-execution](https://github.com/jkgdq789-blip/verify-by-execution-skill) -- Enforce execution-based verification for every code change (reproduce, verify, both paths)

## How to Install a Skill

Copy the SKILL.md file to `~/.claude/commands/<name>.md`, then use `/<name>` in Claude Code.
